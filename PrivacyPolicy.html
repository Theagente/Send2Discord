Privacy Policy

Effective Date: August 19, 2025

Overview

The Send2Discord Chrome extension does not collect, store, or transmit any personal data. All functionality is triggered by the user and operates entirely within the browser.

Permissions Used

  • activeTab: Used to access the URL and content of the currently active tab when you press a hotkey to share a link.
  • scripting: Used to inject a lightweight script that detects which link your mouse is hovering over.
  • storage: Used to save your Discord webhook URL and extension preferences locally on your device.
  • host permissions: Used to access page content so the extension can detect hovered links. This access is only used when you press a hotkey.

Data Collection

We do not collect, transmit, or store any personal information. No data is sent to external servers other than the Discord webhook you configure.

User Control

All actions are user-initiated. You can disable or remove the extension at any time via chrome://extensions.

Contact

If you have questions or concerns, please contact us via the GitHub repository.
